# Cyanurus

Cyanurus is an operating system for ARM.

## Build Dependencies

### Required

* Linux
* gmake
* gcc (arm-none-eabi)
* binutils (arm-none-eabi)
* ruby
* qemu-system-arm
* qemu-img
* mkfs.minix

### Optional

* clang
* gdb (arm-none-eabi)

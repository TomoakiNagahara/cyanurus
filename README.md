# Cyanurus

Cyanurus is a Unix-like operating system for ARMv7-A.

![Screen Capture](https://cloud.githubusercontent.com/assets/928237/11760605/6e2ad110-a0e4-11e5-861e-8686bd7f3cc1.gif)

## Build Dependencies

### Required

* Linux
* gmake
* gcc (arm-none-eabi)
* binutils (arm-none-eabi)
* ruby
* qemu-system-arm
* qemu-img
* mkfs.minix

### Optional

* clang
* gdb (arm-none-eabi)
